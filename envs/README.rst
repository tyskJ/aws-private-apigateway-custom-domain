=====================================================================
動作確認手順
=====================================================================
1. EC2 にログイン
---------------------------------------------------------------------
* `envs` フォルダ配下でデプロイ実施後、 `Outputs` にて `KeyPair作成コマンド` 及び `SSHコマンド` が出力される
* `KeyPair作成コマンド` を実施し、 `envs` フォルダ配下に `KeyPair` を作成
* その後、 `SSHコマンド` を実行し EC2 にログインする

2. `curl` 実行
---------------------------------------------------------------------
.. code-block:: zsh

    curl -L -i -v https://カスタムドメインのFQDN/hello

.. note::

    ``-L``: リダイレクト有効化
    ``-i``: HTTPヘッダを出力に含める
    ``-v``: 処理状況のより詳細や説明を出力
